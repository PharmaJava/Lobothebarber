Conexión con GitHub

Necesario para publicar directamente. Se configura una sola vez.

¿Cómo creo el token? (1 minuto)
  1. Entra en GitHub → Tokens (fine-grained).
  2. En Repository access, elige Only select repositories → este repositorio.
  3. En Permissions → Repository permissions, pon Contents: Read and write.
  4. Genera el token, cópialo y pégalo arriba. Guárdalo en sitio seguro.

⚠️ El token se guarda en este navegador. Úsalo solo en equipos de confianza y revócalo si se filtra.
